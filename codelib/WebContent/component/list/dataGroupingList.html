使用(无需额外的调用JS) 

			<div id="phaseStatusTypeId" name="phaseStatusTypeId" class="AI-Comp"
				type="combox" ds="dimPhaseAndStatus" showstyle="group"
				multisel="false" value="" showall="true" alltxt="--请选择-" allval="-1"></div>

属性说明 

			class		必须为AI-Comp
			ds 			JSON数据ID
			type		组件的类型
						combox	表示下拉框
			allowFilter	是否允许过滤
			showall
			alltxt		未选择时的中文展示
			allval		未选择时的value值
			multisel	是否多选
			showstyle	显示类型
						group 	分组显示
						tree	树形显示
			cascade		联动对应的下级ID
