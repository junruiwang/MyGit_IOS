过滤下拉框--效果展示

使用解释

大体同普通下拉框，当设置div中的allowFilter为true时，开启过滤功能

引用的JS和CSS

下拉数据源(JSON格式) 

			var prjTypeJson = [
				{"val":20001,"pid":-1,"txt":"选项A","enable":0,"leaf":true},
				{"val":20002,"pid":-1,"txt":"选项B","leaf":true},
				{"val":20003,"pid":-1,"txt":"选项C","leaf":true}
			];

使用(无需额外的调用JS) 

			<div id="projectTypeId" name="basePrjInfo.projectTypeId"
				showall="true" class="AI-Comp" ds="prjTypeJson" type="combox"
				allowFilter="true" showall="true" alltxt="--请选择-" allval="-1"></div>

属性说明

			class		必须为AI-Comp
			ds 			JSON数据ID
			type		组件的类型
						combox	表示下拉框
			allowFilter	是否允许过滤
			showall
			alltxt		未选择时的中文展示
			allval		未选择时的value值
			multisel	是否多选
			showstyle	显示类型
						group 	分组显示
						tree	树形显示
			cascade		联动对应的下级ID

方法说明 

                       创建对象        var comtmp = new AI.Comp.Combox({
			                               el:'combox1',					
			                               showAll:true,
			                               ds:arr,//eval([{text:'M902',value:58920140},{text:'T105',value:86202159}])在转化为js数组
			                               multisel : true,
			                               allowfilter : true,
			                               value:arrayDimLevel2Str,
			                               id:'combox1'
		                          });
                       获取对象        AI.getComp('id').getValue()
                       销毁对象        AI.getComp('id').destory()
